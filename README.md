📊 PhonePe Financial Dashboard (Power BI Project)
📌 Overview

This project is a Power BI dashboard designed to analyze financial transactions across multiple services such as Loans, Insurance, Bills, and Transfers. It provides insights into transaction performance, success/failure rates, and service-wise usage.

 Features

📅 Date Range Filter – Analyze data dynamically over time

💰 Total Transaction Amount – Displays overall financial volume

🔢 Total Transactions Count – Shows number of transactions

✅ Success vs Fail Analysis – Quick performance overview

📊 Service-wise Distribution – Compare usage across services

📉 Failure Reasons Analysis – Understand why transactions fail

📈 Monthly Trend (Date vs Amount) – Identify trends over time

📂 Dataset Details

The dataset includes the following key fields:

Booking ID

Booking Status

Customer ID

Vehicle Type / Service Type

Pickup & Drop Locations

Transaction Amount

Ride Distance

Payment Method

Cancellation Reasons

Ratings (Driver & Customer)

📊 Dashboard Insights

Majority of transactions are successful (~4600+)

Very low failure rate (~200 transactions)

Monthly trends show fluctuations in transaction volume

Key failure reasons include:

Insufficient balance

Server errors

Wrong PIN

🛠️ Tools & Technologies

Power BI Desktop

Data Cleaning & Transformation (Power Query)

Data Modeling (Relationships, Measures)

DAX (for calculated measures)

📈 Key Visualizations

Bar Chart → Amount by Service Type

Line Chart → Monthly Transaction Trend

Pie Chart → Failure Reasons

KPI Cards → Total, Success, Fail

📌 How to Use

Download the .pbix file from this repository

Open in Power BI Desktop

Refresh data if needed

Use filters to explore insights
